# pacta.data.scraping

The goal of `pacta.data.scraping` is to provide tools to scrape critical
data sets from web sources and prepare them for input to the PACTA data
preparation process.

## Installation

You can install the development version of pacta.data.scraping from
[R-universe](https://r-universe.dev) with:

``` r
install.packages("pacta.data.scraping", repos = "https://rmi-pacta.r-universe.dev")
```

or from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("RMI-PACTA/pacta.data.scraping")
```
